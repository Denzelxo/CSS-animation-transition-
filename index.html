Theme Switcher

Project Description

This project demonstrates the power of modern web development techniques, showcasing:

  • Smooth CSS transitions and animations
  • Dynamic theme switching with three beautiful themes
  • Local storage for persistent user preferences
  • Responsive design that works on all devices
  • Clean, modern JavaScript implementation

Built with: HTML5, CSS3, JavaScript, and Font Awesome icons
